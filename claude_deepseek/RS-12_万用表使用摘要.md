# RS-12 数字万用表使用摘要

> 编号: 123-1939 | 版本: 001 (2017-09-28)

---

## 一、产品概述

RS-12 是一款 **3½ 位（2000 位）液晶显示** 数字万用表，支持交直流电压、直流电流、电阻、二极管、短路蜂鸣、电池测试及非接触交流电压检测等功能。

| 技术指标 | 参数 |
|---------|------|
| 显示 | 3½ 位，2000 位 LCD，1.1" 数位高度 |
| 输入阻抗 | >1MΩ |
| 交流电压频宽 | 45Hz ~ 450Hz |
| 测量率 | 每秒 2 次 |
| 超量程提示 | 显示 "1" |
| 极性 | 自动（正极无显示，负极显示 "-"） |
| 低电提示 | 显示 "BAT" 符号 |
| 供电 | 9V 电池 (NEDA 1604) × 1 |
| 重量 | 255g |
| 尺寸 | 150mm × 70mm × 48mm |
| 操作环境 | 0°C ~ 50°C，湿度 <70% |
| 储存环境 | -20°C ~ 60°C，湿度 <80% |
| 海拔 | 最高 2000 米（室内使用） |
| 安全认证 | 过电压类别 II，污染级别 2，双绝缘保护 |

## 二、端口与控制面板

### 端口

| 端口 | 说明 |
|------|------|
| **COM** | 负极公共端（黑色表笔），对地电压不超过 500V |
| **V/Ω** | 正极端口（红色表笔），用于电压、电阻、二极管、电池测试 |
| **mA** | 毫安电流测量端口（红色表笔），最大 200mA，带 0.2A/250V 保险丝 |
| **10A** | 大电流测量端口（红色表笔），最大 10A，带 10A/250V 保险丝 |

### 按键

| 按键 | 功能 |
|------|------|
| **HOLD** | 数据保持——按下锁定当前读数并显示 "HOLD" 字符，再按恢复 |
| **BACKLIGHT** | 背光——按下开启 LCD 背光，再按关闭 |

### 功能符号

| 符号 | 含义 |
|------|------|
| `)))` | 蜂鸣/短路通断指示 |
| `→+` | 二极管测试指示 |
| `µ` | 微（电流范围） |
| `m` | 毫（电压/电流范围） |
| `k` | 千（电阻范围） |
| VDC | 直流电压 |
| VAC | 交流电压 |
| ADC | 直流电流 |
| BAT | 电池电量不足 |

## 四、各功能操作指南

### 4.1 直流电压测量 (DCV)

1. 功能转盘置于 **V DC** 档位
2. 黑色表笔插入 **COM** 端口，红色表笔插入 **V** 端口
3. 表笔接触被测物（红正、黑负）
4. 读取显示屏数值；极性接反时显示负号

> 注意：不要在设备开关电源瞬间测量，瞬间高压会损坏仪表。

### 4.2 交流电压测量 (ACV)

1. 功能转盘置于 **V AC** 档位
2. 黑色表笔插入 **COM** 端口，红色表笔插入 **V** 端口
3. 表笔接触被测物
4. 读取显示屏数值；可逐步调低档位获取更高分辨率

> 警告：测量 240V 插座时若读数为 0，可能是表笔未接触到内部金属触点，需检查接触。

### 4.3 直流电流测量 (DCA)

1. 黑色表笔插入 **COM** 端口
2. **≤200mA**：转盘置于 DC mA 档，红色表笔插入 **mA** 端口
3. **≤10A**：转盘置于 10A 档，红色表笔插入 **10A** 端口
4. **断开被测电路电源**，在测量点打开电路绝缘层
5. 黑色表笔接电路负极侧，红色表笔接电路正极侧
6. 接通电源，读取读数

> 警告：10A 档位测量时间不得超过 **30 秒**，超时可能损坏仪表或表笔。mA 测量时可逐步调低档位以获得更高分辨率。

### 4.4 电阻测量 (Ω)

1. **断开电源，电容放电，取出电池**
2. 功能转盘置于最高电阻 Ω 档位
3. 黑色表笔插入 **COM** 端口，红色表笔插入 **Ω** 端口
4. 表笔接触被测元件（建议断开电路一端以避免并联干扰）
5. 读取读数后调低档位以获得更高分辨率

### 4.5 短路蜂鸣测试

1. 功能转盘置于蜂鸣（`)))`）位置
2. 黑色表笔插入 **COM**，红色表笔插入 **Ω** 端口
3. 表笔接触被测物体
4. **电阻 <30Ω 时蜂鸣器发声**；开路时显示 "1"

> 警告：不可在通电状态下进行在线短路蜂鸣测试。

### 4.6 二极管测试

1. 功能转盘置于二极管（`→+`）位置
2. 黑色表笔插入 **COM**，红色表笔插入 **V** 端口
3. 表笔接触二极管两端：
   - **正向**：显示 400mV ~ 700mV
   - **反向**：显示 "1"（超量程）
   - **短路**：显示接近 0V
   - **开路**：双向均显示 "1"

> 测试参数：测试电流最大 1mA，开路电压典型值 2.8V DC。

### 4.7 电池测试

1. 黑色表笔插入 **COM**，红色表笔插入 **V** 端口
2. 选择 **1.5V** 或 **9V** 档位
3. 红表笔接电池正极，黑表笔接电池负极
4. 读取数值

| 电池类型 | 良好 | 较弱 | 坏的 |
|---------|------|------|------|
| **9V** | >8.2V | 7.2V ~ 8.2V | <7.2V |
| **1.5V** | >1.35V | 1.22V ~ 1.35V | <1.22V |

### 4.8 非接触交流电压检测 (NCV)

1. 将探头靠近或插入火线输出插座孔
2. 火线带有 220V 交流电时，指示灯点亮
3. 零火线缠绕在一起时需将其分开后再测试
4. 静电或其他带电体可能导致指示灯闪烁，属正常现象

## 六、安全守则

1. 测量值不得超过量程范围
2. 测量高压电路时严格做好个人及设备安全防护
3. COM 端口对地电压超过 **500V** 时禁止电压测试
4. 电流/电阻/二极管档位下，**禁止将表笔连接电路**，否则损坏仪表
5. 电阻/二极管测试前须将电容放电、断开电源
6. 更换保险丝/电池前须关闭电源、取下表笔
7. 电池盖和保险丝盖须拧紧后方可使用

### 安全标识

| 标识 | 含义 |
|------|------|
| ⚠ 参照说明书 | 该操作须按说明书执行 |
| **WARNING** | 可能出现危险电压，可能导致死亡或严重伤害 |
| **CAUTION** | 可能出现危险电压，可能导致仪表损坏 |
| **MAX 500V** | 禁止连接到 500VAC/VDC 的电路 |
| ⚡ 端口标记 | 该端口可能出现危险电压 |
| 回（双绝缘） | 双绝缘保护 |

## 七、维护

### 更换电池

1. BAT 符号出现时需更换
2. 断开表笔，用螺丝刀拧开后盖
3. 安装 9V 电池（注意正负极）
4. 盖好后盖并拧紧

### 更换保险丝

- **mA/µA 量程**：0.2A/250V 快速熔断保险丝
- **10A 量程**：10A/250V 快速熔断保险丝

1. 断开表笔与被测物
2. 拧开保险丝盖板
3. 取出旧保险丝，装入同规格新保险丝
4. 盖好并拧紧

> 警告：更换后盖未盖紧前禁止操作仪表。

## 八、常见问题

| 现象 | 原因 / 处理 |
|------|------------|
| 显示 "OL" | 测量值超出当前档位，切换至更高档位 |
| 显示 "BAT" | 电池电压不足，需更换电池 |
| 低档位空载数值跳动 | 输入灵敏度高导致，接通电路后即稳定 |
| 仪表工作不正常 | 检查保险丝和电池是否完好、安装是否正确 |
| 插座有电但读数为 0 | 表笔未接触到插座内部金属接口 |
